# Aurora
An extremely minimal kernel for x86_64 systems made in C.

## Usage
Boot the aurora.iso file, and you should get a console with 45 usable commands for
basic file management, directory switching and more.

## Disclaimer
This hasn't been properly tested, I reccomed first trying to launch in a Virtual
Machine, as I don't know if this can boot consistently.
